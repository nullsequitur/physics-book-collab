## Εισαγωγή

Το παρόν repository δημιουργήθηκε με στόχο την προσομοίωση ομαδικής συγγραφής
ενός συνεργατικού επιστημονικού κειμένου, στο πλαίσιο του μαθήματος **"Νέες
Τεχνολογίες στη Διδακτική των Φυσικών Επιστημών"** του Τμήματος Φυσικής του
Πανεπιστημίου Ιωαννίνων, για το ακαδημαϊκό έτος 2025-2026.

Στόχος είναι η συλλογή κεφαλαίων από τη διδαχθείσα ύλη Φυσικής Λυκείου με χρήση
σύγχρονων εργαλείων συγγραφής (LaTeX) και υπολογιστικών μεθόδων (Python).

Κάθε φοιτητής/τρια θα δημιουργήσει ένα κεφάλαιο και θα το ενσωματώσει στο
υπόλοιπο [βιβλίο](.book/physics-collab.pdf). Για τη συγγραφή θα αντλήσετε υλικό
από τα σχολικά βιβλία Φυσικής Γενικού Λυκείου, τα οποία είναι διαθέσιμα
[online](https://ebooks.edu.gr/ebooks/).

---

## Ζητούμενα Εργασίας

Κάθε φοιτητής/τρια αναλαμβάνει ένα θέμα Φυσικής και οφείλει να:

1.  Συγγράψει ένα κεφάλαιο (2-3 σελίδες) σε LaTeX. Το κείμενο πρέπει:
    - Να είναι επιστημονικά ακριβές, αναδιατυπώνοντας τη σχολική ύλη.
    - Να περιέχει μαθηματικές εξισώσεις και cross-references.
2.  Δημιουργήσει και να ενσωματώσει τουλάχιστον ένα επιστημονικό γράφημα.
    - Το γράφημα πρέπει να παραχθεί προγραμματιστικά (με Python/Matplotlib).
    - Ο κώδικας Python που παράγει το γράφημα πρέπει να συμπεριληφθεί στο
      repository.

## Οδηγίες Συμμετοχής

Ακολουθήστε τα παρακάτω βήματα για να υποβάλετε την εργασία σας.

### Κλωνοποίηση (Clone)

Κατεβάστε το repository στον υπολογιστή σας:

```bash
git clone https://github.com/nullsequitur/physics-book-collab.git
```

και μεταβείτε στον φάκελο

```bash
cd physics-book-collab
```

### Δημιουργία νέου κλάδου (branch)

**ΠΡΟΣΟΧΗ:** Δεν εργαζόμαστε ποτέ στο main branch. Δημιουργήστε ένα νέο branch
με τον Αριθμό Μητρώου σας (π.χ. αν το ΑΜ είναι 1234):

```bash
git checkout -b ph01234
```

### Προσθήκη αρχείων

Προσθέστε τα αρχεία που θα δημιουργήσετε στον κατάλληλο φάκελο με κατάλληλη
ονομασία. Συγκεκριμένα:

- /book/chapters/ph01234.tex
- /python-scripts/ph01234.py
- /book/figures/ph01234.png

### Αποθήκευση αλλαγών

```bash
git add ta_arxeia_mou
```

όπου κάνετε αντικατάσταση του placeholder `ta_arxeia_mou` με το σωστό όνομα/path
των αρχείων σας.

Π.χ., αν βρίσκεστε στον φάκελο `physics-book-collab`:

```bash
git add book/chapters/ph01234.tex python-scripts/ph01234.py book/figures/ph01234.png
```

Έπειτα κάνετε `commit` τις αλλαγές σας, π.χ.:

```bash
git commit -m "ph01234: added figure and python script"
```

και τέλος ανεβάζετε το branch σας στο GitHub:

```bash
git push -u origin ph01234
```

### Προαπαιτούμενα

Για την τοπική εκτέλεση του κώδικα και τη συγγραφή του βιβλίου, απαιτούνται:

- **Python 3.8+** με εγκατεστημένη τη βιβλιοθήκη `matplotlib`
- **LaTeX Distribution** (π.χ. TeX Live, MiKTeX ή MacTeX)

Αναλυτικές οδηγίες και tutorials για όλα τα παραπάνω μπορείτε να βρείτε στη
[σελίδα e-course](https://ecourse.uoi.gr/course/view.php?id=1958) του μαθήματος.
